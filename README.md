# Vimosanan — Portfolio

Personal portfolio site built with Vite + React, deployed to GitHub Pages.

## Tech Stack

- [Vite](https://vitejs.dev/) — build tool
- [React](https://react.dev/) — UI framework
- [Framer Motion](https://www.framer.com/motion/) — animations
- [Tailwind CSS](https://tailwindcss.com/) — utility styles
- [gh-pages](https://github.com/tschaub/gh-pages) — deployment

## Getting Started

```bash
# Install dependencies
make install

# Start development server
make dev
```

## Commands

| Command        | Description                          |
|----------------|--------------------------------------|
| `make install` | Install dependencies                 |
| `make dev`     | Start development server             |
| `make build`   | Build for production                 |
| `make preview` | Preview production build locally     |
| `make deploy`  | Build and deploy to GitHub Pages     |
| `make clean`   | Remove dist folder                   |
| `make ship`    | Install, build and deploy in one step|
